Integration Instructions

Stripe Checkout Setup

To integrate Stripe Checkout for real payments:

  1. Sign up for a Stripe account at stripe.com
  2. Create products and pricing tiers in your Stripe Dashboard
  3. Get your publishable and secret API keys
  4. Replace the placeholder keys in the JavaScript below with your test keys
  5. Implement server-side checkout session creation
  6. Test with Stripe test cards (e.g., 4242 4242 4242 4242)
  7. Once tested, replace with production keys for live payments
